# 安装
安装hexo-cli
```shell
$ npm install -g hexo-cli
```
初始化blog项目
```shell
$ hexo init <folder>
$ cd <folder>
$ npm install
```
生成静态文件
```shell
$ hexo generate # -d 文件生成后立即部署网站 -w 监视文件变动
```
启动服务器。默认情况下，访问网址为： http://localhost:4000/
```shell
$ hexo server
```
详细文档请戳[这里](https://hexo.io/zh-cn/docs/)
